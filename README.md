# 📊 Proyecto de Análisis de Datos (Bootcamp)

## 📌 Descripción
Este repositorio contiene un proyecto educativo desarrollado como parte de un bootcamp de análisis de datos. Su objetivo principal es aplicar conceptos fundamentales de análisis, procesamiento y visualización de datos utilizando Python.

Actualmente, el proyecto se encuentra en fase de desarrollo y no cuenta aún con un conjunto de datos definido.

## 🎯 Objetivos
- Practicar habilidades de análisis de datos
- Aplicar técnicas de limpieza y transformación de datos
- Explorar visualización de datos
- Construir un flujo de trabajo reproducible en Python

## 🛠️ Tecnologías utilizadas
- Python
- Jupyter Notebook (opcional)
- Librerías comunes (en proceso de definición):
  - pandas
  - numpy
  - matplotlib / seaborn

## ⚙️ Requisitos
Asegúrate de tener instalado:
- Python 3.8 o superior
- pip o conda
